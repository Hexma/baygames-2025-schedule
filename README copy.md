# 🏅 2025年中国全运会 - 金牌赛赛程

一个为手机端设计的Web应用，展示2025年中国全运会（全运会）的金牌赛赛程。

## ✨ 主要功能

✨ **核心特性：**
- 📱 完全响应式的手机设计
- 🏆 多个运动项目的金牌赛事赛程
- 🔍 按运动项目分类筛选赛事
- 📅 按日期组织的赛程信息
- ⏰ 赛事时间和场馆信息
- 🥇 已完成赛事的金牌得主和成绩显示
- 🎨 美观现代的UI设计，流畅的交互体验

## 📋 支持的运动项目

- 田径
- 游泳
- 羽毛球
- 网球
- 排球
- 篮球
- 跳水
- 乒乓球
- 体操
- 举重
- 赛艇
- 射击
- 自行车
- 柔道
- 跆拳道

## 🚀 安装与运行

### 环境要求
- Node.js 14.0 或更高版本
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm start
```

应用将在浏览器中打开 `http://localhost:3000` 并在您修改代码时自动刷新。

### 生产构建

```bash
npm run build
```

在 `build/` 文件夹中创建优化的生产版本。

## 📖 使用说明

1. **查看赛程**：主页显示所有按日期组织的金牌赛事
2. **筛选运动项目**：使用顶部的运动项目按钮进行筛选
3. **查看赛事详情**：每个赛事卡片显示：
   - 赛事名称和运动项目类别
   - 赛事状态（已完成、待进行、进行中、已取消）
   - 开始时间和场馆
   - 金牌得主和成绩（仅已完成赛事）

## 🔴 赛事状态说明

| 状态 | 颜色 | 含义 |
|-----|------|------|
| 已完成 | 🟢 绿 | 赛事已结束，有最终成绩 |
| 待进行 | 🔵 蓝 | 即将开始的赛事 |
| 进行中 | 🟠 橙 | 正在进行的赛事 |
| 已取消 | ⚫ 灰 | 已被取消的赛事 |
